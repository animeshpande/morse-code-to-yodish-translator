Language used: python3.6.4 (Download from https://www.python.org/downloads/release/python-364/)

Encoding:
dot: .
dash: -
character separator: <space>
word separator: /

Running the python script:
After the above version of python is downloaded, the script can be run through the command line on windows, linux and mac.
On running the script, it will prompt you to enter the morse code to be translated.
The morse code can be input with the above specified encoding. There is no support for punctuation marks.
example input:
Please enter the morse code: - .... . / -- .. -. -.. / --- ..-. / .- / -.-. .... .. .-.. -.. / .. ... / - .-. ..- .-.. -.-- / .-- --- -. -.. . .-. ..-. ..- .-..

The output will be an ascii figure of yoda with the decoded morse code sentence displayed in yodish (yoda-speak)
example output:
 ------------------------------------------
<Truly wonderful, the mind of a child is.  >
